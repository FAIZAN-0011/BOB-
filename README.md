## BOB-

An AI chatbot built using Streamlit and Groq API.

## Live Demo

https://zexhst9t6w5rzkf55ngyvr.streamlit.app/

## Features

- AI-powered chatbot
- Streamlit UI
- Conversation history
- Groq LLM integration

## Installation

pip install -r requirements.txt

## Run Locally

streamlit run app.py
